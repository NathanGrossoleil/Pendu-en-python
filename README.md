# Pendu-en-python
Jeu du pendu réalisé en python
Ce projet est un jeu du pendu réalisé à 2 en python qui comporte une partie grafique quand le joueur se trompe.
Dans ce jeu il ya différentes fonctionnalitées:
  - Choix aléatoire d'un mot à deviner
  - Saisie et vérification des lettres par l'utilisateur
  - Gestion du nombre d'essais
  - Détection de la victoire ou de la défaite
